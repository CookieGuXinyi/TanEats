# TanEats - 校园周边摊贩美食小程序

TanEats 是一款面向校园周边小吃摊贩和学生用户的美食发现与互动平台。当前版本完成了前端静态 UI 框架搭建和微信云开发环境的初步集成，具体功能正在逐一推进中。

## 当前版本功能

**基础功能**
- 八个核心页面的静态 UI 框架：发现页、地图视图、消息中心、个人资料、用户注册、商家仪表板、AI 聊天、摊位地图
- 微信云开发环境配置
- 用户、摊位申请、商铺、产品数据库集合（`users`, `stall_applications`, `stalls`, `products`）创建
- 手机号注册与登录功能（当前为模拟验证码发送）

**摊主端**
- 摊主商铺注册功能（当前为手动后台审核）
- 商铺信息修改功能
- 商品增删改查与上架状态切换功能
- 个人资料页面与摊主工作台动态展示真实用户数据

**用户端**
- 发现页“今日热门”接入真实评论数据（当前按发布时间排序）
- 写评论页与评论详情页，支持多店铺关联和自定义话题标签
- 评论贴搜索页面，支持关键词搜索、历史记录、热门推荐（静态模拟数据）、筛选排序
- 无搜索时默认显示最新评论，支持滚动分页加载
- 店铺详情页（面向用户），与真实数据库对接

## 待完善问题
- 搜索范围暂限于 content 字段，需扩展至店铺名、标签、用户名等
- 搜索需实现更广泛的模糊搜索逻辑及合理的综合关联度评分排序
- 发现页评论排序同上需优化（支持评分、热度等多维度）
- 验证码需接入真实短信服务
- 摊主申请需开发审核员管理页面
- AI 问答需接入真实接口
- 摊位地图页面待完善功能开发（AI-ver now）
- 摊位商铺搜索页面待完善功能开发（AI-ver now）
- 跨摊拼单功能待开发

## 技术栈

- 微信小程序原生框架（WXML + WXSS + JavaScript）
- 微信云开发（云函数 + 云数据库）
- Git 版本管理
